Lab 15: Preguntas



¿Por qué es una buena práctica separar el modelo del controlador?


Porque permite separar los componentes de nuestra aplicación con cierto grado de modularidad, y a la vez, nos otorga un mayor control para seguridad.

¿Qué es SQL injection y cómo se puede prevenir?


Es la inserción de código SQL en campos de input de usuario con intención de obtener información o realizar acciones que pueden perjudicar al sistema. Se puede prevenir realizando validaciones de información enviada a través de input de usuario, y no realizar queries con input de usuarios.
